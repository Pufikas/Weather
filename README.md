# How to use

 Get the demo server and api from.
 
 Get API `https://openweathermap.org/api`
 
 Server demo `https://cors-anywhere.herokuapp.com/corsdemo`

Install packages `npm i`

Create key.js and import your apikey as following
```js
APIkey="yourkeyhere"
```
